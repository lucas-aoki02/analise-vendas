Olá! Bem vindo ao projeto análise de vendas

- Esse projeto tem como objetivo gerar análises dos dados das vendas de uma empresa fictícia vendedora de carros.

- O arquivo base.sql contém o código que alimenta o banco de dados.

- O modelo transacional do banco de dados usado no projeto é ilustrado no arquivo diagrama_transacional.PNG.

- A dashboard das análises está no arquivo excel dashboard_vendas.xlsx. No Excel há uma planilha para a 
  dashboard, uma planilha para o input dos dados das queries, e uma planilha que traz o código de cada query.

- O projeto foi realizado como atividade do curso SQL para Análise de Dados: Do básico ao avançado.
  link: https://www.udemy.com/course/sql-para-analise-de-dados/


 Hello! Welcome to the sales analysis project

- This project aims to generate analysis of sales data from a fictitious car sales company.

- The base.sql file contains the code that feeds the database.

- The database transactional model used in the project is illustrated by the diagram_transacional.PNG file.

- The analysis dashboard is in the excel file dashboard_vendas.xlsx. In Excel there is a spreadsheet for
   dashboard, a spreadsheet for inputting query data, and a spreadsheet that shows the code of each query. 

- The project was carried out as an activity in the SQL para Análise de Dados: Do básico ao avançado course
  link: https://www.udemy.com/course/sql-para-analise-de-dados/
